OVASI

Objeto Virtual de Aprendizaje para Sentencias Iterativas

BIENVENIDOS

Instrucciones

Estimado estudiante, con la presente plataforma se quiere apoyar el proceso de aprendizaje para el tema de SENTENCIAS ITERATIVAS (Son aquellas que controlan la repetición de un conjunto de instrucciones denominado bloque o cuerpo del bucle, mediante la evaluación de una condición o mediante un contador.) de la materia Fundamentos de Programación.

Este Objeto Virtual de Aprendizaje consta de 3 temas principales en la sentencias iterativas: FOR, WHILE, DO WHILE; cada tema se aborda con una definición, ejemplos y ejercicios que usted debe desarrollar y autoevaluarse para verificar en qué proceso está fallando.

Es necesario que usted se registre para que pueda presentar la evaluación final y así se puedan ir guardando los puntos que alcanza cada vez que realice la prueba, esta se podrá realizar solo en 3 intentos.
